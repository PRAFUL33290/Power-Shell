Astuces pour passer de Linux à Windows

01

Ouvre PowerShell, pas CMD

Win + X → Terminal Windows ou PowerShell. C’est l’équivalent moderne de ton terminal Linux.

02

Les alias Linux existent déjà

Dans PowerShell, ls, cd, pwd, cat, mkdir, rm fonctionnent souvent — ce sont des alias vers des cmdlets.

03

Les chemins changent

/ devient \ (ou / marche aussi en PS). ~ = $HOME. /etc n’existe pas : regarde C:\Windows\System32.

04

Objets, pas du texte

Le pipeline PowerShell passe des objets, pas des lignes de texte. Get-Process | Where-Object CPU -gt 10.

05

Git Bash & WSL

Si tu veux du vrai Bash sous Windows : installe Git for Windows ou WSL2 (wsl --install).

06

Aide intégrée

Get-Help Get-ChildItem -Full · Get-Command *Item* · Get-Alias pour lister les raccourcis.
