Astuces pour passer de Linux à CMD Windows

01

Ouvre l’invite de commandes

Recherche Windows → Invite de commandes, ou Win + R puis tape cmd et Entrée.

02

Les chemins sont différents

Sous Linux : /home/user. Sous Windows : C:\Users\user. Les dossiers se séparent avec \.

03

Pas de ~ ni de $VAR

Le dossier perso = %USERPROFILE%. Une variable = %NOM% (ex. %PATH%, %CD%).

04

dir ≈ ls, type ≈ cat

Beaucoup d’équivalents sont simples : dir, cd, copy, move, del, type, cls.

05

Aide sur une commande

Tape commande /? — ex. dir /? ou copy /?. C’est l’équivalent de man / --help.

06

Admin si besoin

Clic droit sur « Invite de commandes » → Exécuter en tant qu’administrateur (équivalent de sudo).
