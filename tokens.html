01

Entrée ≠ sortie

Le prompt (entrée) est souvent moins cher que la réponse générée (sortie). Les agents qui “parlent beaucoup” coûtent plus cher.

02

1M tokens, c’est quoi ?

Environ 750 000 mots en anglais. Une page web ≈ 500–2 000 tokens. Un long chat peut vite monter.

03

Éco vs frontier

GPT-4o-mini et Haiku pour le volume. Grok 4.5 / Sonnet / GPT-4.1 pour la qualité et le code.

04

Bundle vs pay-as-you-go

Vibe facture au mois (plusieurs modèles inclus). Les API Grok / OpenAI / Claude facturent au token consommé.

05

Formule simple

coût = (entrée/1M)×prix_in + (sortie/1M)×prix_out

06

Vérifie toujours

Les tarifs bougent. Cette page reflète le tableau du 17/07/2026 — croise avec les pages officielles.
