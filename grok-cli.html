Pourquoi les devs adoptent Grok CLI

Pas de bascule d’onglets. Pas de copier-coller permanent. Ton workflow reste dans le terminal — là où le code se construit déjà.

TUI full-screen, pensée pour le code

Scrollback live des outils, prompt en bas, pièces jointes avec @fichier, sessions sauvegardées. Tu vois chaque read, edit et commande en temps réel.

Agent, pas chatbot

Explore le repo, planifie, édite, relance les tests. Multi-tours jusqu’à ce que le job soit fait.

Mode headless

grok -p "…" pour scripts, CI, cron. Sortie plain, JSON ou streaming.

MCP servers

Branche GitHub, calendrier, bases de données… Les outils externes deviennent des armes de l’agent.

Skills & plugins

Encode tes procédures en SKILL.md. Plus besoin de tout réexpliquer à chaque session.

Permissions & sandbox

Approuve commande par commande, ou YOLO. Sandbox FS/réseau pour limiter la surface d’attaque.

Subagents & plan mode

Délègue en parallèle, isole en worktree, planifie avant d’écrire du code en masse.

IDE via ACP

Mode agent stdio / WebSocket pour brancher Zed, Neovim, Emacs ou ton outil maison.

macOS · Linux · Windows

Install bash ou PowerShell. WSL, Git Bash, terminal natif — le CLI suit ton OS.

Les mérites, en clair

  • Vélocité Tu restes en flow : moins d’alt-tab, plus de shipping.
  • Contrôle Permissions granulaires, sandbox, allow/deny rules — pas une boîte noire incontrôlable.
  • Mémoire de projet Sessions, rules (AGENTS.md), skills d’équipe : l’agent apprend ton contexte.
  • Puissance modèle Propulsé par les modèles Grok (xAI) — fort sur code, agents et raisonnement long.
  • Écosystème ouvert MCP, ACP, hooks, plugins, thèmes — tu étends sans attendre une feature maison.
  • Cross-platform Linux, macOS, Windows (PowerShell / Git Bash / WSL). Une seule commande d’install.

Install en une ligne

macOS / Linux / Git Bash

curl -fsSL https://x.ai/cli/install.sh | bash

Windows PowerShell

irm https://x.ai/cli/install.ps1 | iex

Ensuite :

  1. grok --version — vérifier l’install
  2. grok — lancer le TUI (auth navigateur la 1re fois)
  3. Ou export XAI_API_KEY=… pour la CI / headless
  4. grok update — rester à jour

Arrête de copier-coller. Commence à déléguer.
